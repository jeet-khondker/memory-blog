How To Setup Your Profile Picture?

If you are seeing an anonymous profile picture after your registration and login to MemoryBlog, then it means you are not a registered user in Gravatar. Please register to Gravatar and use that profile picture globally to MemoryBlog.

To create a Gravatar account, please access this link: 👉🏻 Gravatar Registration. You just need the email address, username and password for registration.

Simply, use the same email address for both Gravatar account and MemoryBlog Web Application during registration.
